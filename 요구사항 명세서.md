## 1. 프로젝트 소개  
  
### 1.1 주제 선정 이유  
  
요즘 뉴스를 보면 “기록적인 폭염” “30년만의 한파”라는 말을 자주 듣게 된다. 극단적인 날씨 변화가 점점 많아지고 있으며, 이러한 이상 기온을 미리 알 수 있다면 건강 관리와 재해 예방에 도움이 될 것이다.  
  
### 1.2 목표  
  
이 프로젝트의 최종 목표는 다음과 같다:  
  
1. **인터넷 API를 통한 자동 데이터 수집** → 서울의 과거 1년 기온 데이터 수집  
2. **머신러닝 알고리즘으로 이상기온 탐지** → 컴퓨터가 폭염과 한파를 자동으로 감지  
3. **폭염(>35°C)과 한파(<10°C) 분류** → 이상 기온이 어떤 종류인지 판별  
4. **결과를 저장하고 그래프로 시각화** → 누구나 쉽게 이해할 수 있도록 표현  
  
**1.3 사용 기술**

| 주요 라이브러리 | pandas, numpy, matplotlib, scikit-learn |

## 4. 프로그램 설명  
  
### 4.1 1단계: 데이터 수집 (collect_data.py)  
  
```
import requests
from datetime import datetime, timedelta

# 오늘 날짜부터 1년 전까지의 기온을 요청
end_date = datetime.now().date()
start_date = end_date - timedelta(days=365)

# Open-Meteo API에 요청
url = "https://archive-api.open-meteo.com/v1/archive"
params = {
    "latitude": 37.5665,      # 서울의 위도
    "longitude": 126.9780,    # 서울의 경도
    "start_date": start_date,
    "end_date": end_date,
    "daily": "temperature_2m_max"  # 일별 최고기온
}

response = requests.get(url, params=params)
data = response.json()


```
  
  
  
**이 코드가 하는 일:**  
  
- ==requests== 라이브러리로 인터넷에서 데이터를 가져온다  
- 서울의 좌표(위도, 경도)를 API에 보낸다  
- 1년 동안의 매일 최고기온을 받는다  
- JSON 형식으로 응답이 온다

### 4.3 3단계: 머신러닝으로 이상탐지 (detect.py)  
  
```
from sklearn.ensemble import IsolationForest
import pandas as pd

# 저장한 CSV 파일 읽기
df = pd.read_csv("weather.csv")

# 기온 데이터만 뽑기
X = df[["max_temp"]].values

# Isolation Forest 머신러닝 모델 만들기
iso_forest = IsolationForest(
    contamination=0.05,   # 데이터의 약 5%가 이상치일 것으로 예상
    random_state=42,      # 재현성을 위한 고정값
    n_estimators=100      # 100개의 의사결정 트리 사용
)

# 모델 학습 및 이상치 판정
anomaly_labels = iso_forest.fit_predict(X)
anomaly_scores = iso_forest.score_samples(X)


```
  
  
  
**Isolation Forest가 하는 일 (원리):**  
  
1. **정상적인 데이터들은 뭉쳐있다** → 예: 15°C ~ 30°C 범위  
2. **이상한 데이터는 따로 떨어져있다** → 예: 38°C (폭염)  
3. **알고리즘이 트리(나무)를 만들어서 데이터를 분할한다**  
4. **이상한 데이터는 적은 분할로 고립된다** → 이것이 이상치!  
  
이런 방식으로 라벨(답)이 없어도 자동으로 이상치를 찾을 수 있다.  
  
### 4.4 4단계: 폭염/한파 분류  
  
```
def classify_temperature(max_temp):
    """기온값에 따라 분류하는 함수"""
    if max_temp > 35:
        return "폭염 (Heat Wave)"  # 35°C 이상은 폭염
    elif max_temp < 10:
        return "한파 (Cold Wave)"   # 10°C 이하는 한파
    else:
        return "정상"               # 그 외는 정상

# 각 기온값을 분류
df["type"] = df["max_temp"].apply(classify_temperature)


```
  
  
  
**분류 기준:**  
  
- **폭염**: 35°C 이상 (매우 더움, 건강에 위험)  
- **한파**: 10°C 이하 (매우 추움, 동상 위험)  
- **정상**: 10°C ~ 35°C 사이 (일반적인 기온)  
  
### 4.5 5단계: 결과 저장  
  
```
# 분석 결과를 모두 정리해서 새로운 CSV 파일로 저장
results_df = pd.DataFrame({
    "date": df["date"],                    # 날짜
    "max_temp": df["max_temp"],            # 최고기온
    "is_anomaly": (anomaly_labels == -1),  # 이상 여부 (True/False)
    "anomaly_score": anomaly_scores,       # 이상도 점수 (-1.0 ~ 1.0)
    "type": df["type"]                     # 폭염/한파/정상
})

results_df.to_csv("weather_with_anomalies.csv", index=False)


```
  
  
  
**저장된 파일의 예:**  
  
```

2025-05-06,22.5,False,-0.523,정상
2025-07-15,35.8,True,0.812,폭염 (Heat Wave)
2026-01-10,5.2,True,-0.684,한파 (Cold Wave)
```

### 4.6 6단계: 그래프로 시각화 (visualize.py)  
  
```
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(16, 8))

# ① 정상 기온을 파란 선으로 표시
normal_mask = ~df["is_anomaly"]
ax.plot(df.loc[normal_mask, "date"], 
        df.loc[normal_mask, "max_temp"], 
        "b-", label="정상 기온", linewidth=1.5)

# ② 이상 기온을 빨간 점으로 표시
anomaly_mask = df["is_anomaly"]
ax.scatter(df.loc[anomaly_mask, "date"], 
           df.loc[anomaly_mask, "max_temp"], 
           color="red", s=100, label="이상 기온", zorder=5)

# ③ 7일 이동평균 (추세선) 주황색으로 표시
moving_avg = df["max_temp"].rolling(window=7, center=True).mean()
ax.plot(df["date"], moving_avg, 
        "orange", linestyle="--", linewidth=2, alpha=0.7, label="7일 이동평균")

# ④ 기준선 추가
ax.axhline(35, color="red", linestyle=":", alpha=0.5, label="폭염 기준(35°C)")
ax.axhline(10, color="blue", linestyle=":", alpha=0.5, label="한파 기준(10°C)")

# 그래프 꾸미기
ax.set_title("🌡️ 서울 온도 이상탐지", fontsize=16, fontweight="bold")
ax.set_xlabel("날짜", fontsize=12)
ax.set_ylabel("최고기온(°C)", fontsize=12)
ax.legend(loc="upper right", fontsize=10)
ax.grid(True, alpha=0.3)

# 고해상도(300 DPI)로 저장
plt.savefig("temperature_analysis.png", dpi=300, bbox_inches="tight")
plt.show()


```
  
  
  
**그래프의 구성:**  
  
- 파란선: 정상 기온의 변화 추세  
- 빨간 점: 이상으로 탐지된 기온  
- 주황 점선: 7일간의 평균 기온 (추세를 보기 쉽게)  
- 빨간 점선: 폭염 기준(35°C)  
- 파란 점선: 한파 기준(10°C)

## 10. 배운 점과 느낀 점  
  
이 프로젝트를 통해 배운 것들:  
  
### 10.1 기술적 학습  
  
1. **API 활용**: 인터넷에서 실시간으로 데이터를 가져오는 방법  
2. **머신러닝**: 라벨 없이 자동으로 패턴을 찾는 알고리즘의 원리  
3. **데이터 분석**: 수집한 데이터를 정제하고 분석하는 과정
